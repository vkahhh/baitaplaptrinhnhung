Bài tập buổi 3 gồm:
1. Tạo file filesystem.txt và ghi một đoạn dự liệu bất kì. Thực hiện ghi ra màn hình các thông tin: Loại file,  thời gian chỉnh sửa file lần cuối, kích thước.
2. Viết một chương trình thực hiện ghi giá trị từ 1-1000000 vào hai file riêng fileA.txt và fileB.txt. Tính toán thời gian thực hiện.
3. Viết một chương trình A tạo ra một tiến trình con B rồi in ra PID và PPID của chúng.
4. Viết một chương trình A tạo ra một tiến trình con B. Bên trong A thực hiện tạo ra file hello.txt nằm ở đường dẫn ./log/hello.txt và ghi thông điệp vào file đấy. Ở B sẽ thực hiện đọc lại nội dung file và in ra màn hình.
